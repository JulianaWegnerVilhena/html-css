Negrito / Destaque

Nesta frase temos um termo em negrito usando uma tag B(não semântica)

Nesta frase temos um termo em negrito usando uma tag STRONG (Semântica)

Italico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica)

Nesta frase temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos temabém criar um texto marcado usando a tag MARK.

Usando o modelo acima de como usar mark criar um texto marcado todos os textos estaram da mesma d]cor da marcação.

Podemos temabém mudar a cor do texto marcado usando um texto marcado usando a tag MARK.
